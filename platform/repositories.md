# Platform Repositories

本仓库只对应 `devflow-app-service`。

## 当前边界

- `Project` 和 `Application` 的元数据 CRUD
- `Application.active_manifest` 绑定语义
- 基础设施通过 `devflow-service-common` 提供

## 不再保留的历史说明

- 不再维护 `Configuration`、`Manifest`、`Job`、`Intent`、`Verify` 的仓库级职责说明
- 不再描述 monorepo 拆分阶段的导出策略
